# Trainer Guide

- **Dashboard** (`/trainer`): class overview, leaderboard, alerts, upcoming tasks.
- **Learners** (`/trainer/learners`): search/filter; open a profile for skill radar, voice
  analytics, XP/streak/certs, AI recommendations, feedback, and private notes.
- **Assignments** (`/trainer/assignments`): assign paths/modules/simulations/quizzes/voice
  practice with due date, priority, difficulty; track completion.
- **Cohorts** (`/trainer/cohorts`): compare classes/groups/batches.
- **Analytics** (`/trainer/analytics`): at-risk learners, top performers, velocity.
- **Reports** (`/trainer/reports`): generate + export CSV (PDF/Excel architecture-ready).
- **Notifications** (`/trainer/notifications`): assignments, deadlines, feedback, alerts.
Give feedback with a rating + action items; learners acknowledge it.
